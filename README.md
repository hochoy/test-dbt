### Instructions for test_run.js:
These are the steps taken to create this repo:

dbt version: 0.15.2

```
brew update
brew tap fishtown-analytics/dbt
brew install dbt
dbt init ./test-dbt

cd ./test-dbt

echo "{% macro macro1(input) %}
  SELECT {{ input }}
{% endmacro %}" > macros/macro1.sql

// create test_run.js
touch test_run.js
npm i axios uuid

// run test_run.js
dbt rpc
node test_run.js
```
